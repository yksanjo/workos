# WorkOS Screenshots

This directory contains screenshots and visual documentation of the WorkOS application.

## Screenshot Guidelines

### Recommended Screenshots

1. **Dashboard/Home Page**
   - Main landing page
   - User dashboard after login
   - File: `01-dashboard.png` or `01-home.png`

2. **Board View**
   - Kanban board with items
   - Different status columns
   - File: `02-board-view.png`

3. **Item Details**
   - Item detail modal/page
   - Comments and activity
   - File: `03-item-details.png`

4. **Workflow Management**
   - Workflow list view
   - Workflow creation/editing
   - File: `04-workflow-management.png`

5. **Analytics Dashboard**
   - Analytics overview
   - Charts and metrics
   - File: `05-analytics-dashboard.png`

6. **Automation Builder**
   - Automation creation interface
   - Trigger and action configuration
   - File: `06-automation-builder.png`

7. **AI Optimization Suggestions**
   - Workflow optimization recommendations
   - Bottleneck detection
   - File: `07-ai-optimizations.png`

8. **Mobile View** (if available)
   - Mobile responsive design
   - Mobile app screenshots
   - File: `08-mobile-view.png`

## Screenshot Specifications

- **Format:** PNG or JPG
- **Resolution:** Minimum 1920x1080 for desktop, 1080x1920 for mobile
- **Naming:** Use descriptive names with numbers for ordering
- **File Size:** Optimize images (keep under 1MB when possible)
- **Privacy:** Blur or remove any sensitive data before committing

## Adding Screenshots

1. Take screenshots of the running application
2. Save them in this directory with descriptive names
3. Update the main README.md to reference the screenshots
4. Commit and push to the repository

## Tools for Screenshots

- **macOS:** Cmd + Shift + 4 (select area), Cmd + Shift + 3 (full screen)
- **Windows:** Snipping Tool or Win + Shift + S
- **Linux:** Screenshot tool or `gnome-screenshot`
- **Browser Extensions:** Full Page Screen Capture, Awesome Screenshot

## Image Optimization

Before committing, optimize images:

```bash
# Using ImageMagick (if installed)
convert screenshot.png -quality 85 -resize 1920x1080 screenshot-optimized.png

# Or use online tools like TinyPNG
```
